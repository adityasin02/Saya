# **App Name**: VibeFlow

## Core Features:

- Reel-Style Home Feed: Display songs as swipeable 'music reels' with album art, waveform animation, and play/pause/like/next controls.
- Local Library Display: Show all local songs with sorting options and import functionality.
- AI-Powered Smart Queue: Generate a dynamic 'Smart Queue' based on play count, skip patterns, time of day, and liked genres/artists. LLM is a tool for making informed choices on song order based on the userâ€™s current vibe.
- Favorites and Playlist Curation: Display liked songs and allow custom playlist creation, with AI-powered suggestions for playlist order.
- Settings Customization: Provide settings for theme toggle, AI personalization, equalizer presets, and offline mode.
- Offline Audio Playback: Enable offline playback of local songs, allowing music to be played even without an internet connection.

## Style Guidelines:

- Primary color: Electric purple (#BE29EC) to give off an exciting and modern aesthetic.
- Background color: Dark gray (#26282B) provides contrast and a sleek, modern look.
- Accent color: Neon blue (#2D9CDB) for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif font, offers a clean, neutral and machined look.
- Vertical scrolling feed layout for reels, mimicking popular video-sharing apps.
- Minimal and modern icons for play/pause, like, shuffle/next, and settings.
- Smooth fade-in/fade-out transitions between songs and animated background gradients.
